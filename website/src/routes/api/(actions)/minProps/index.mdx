# minProps

Creates a min properties validation action.

```ts
const Action = v.minProps<TInput, TRequirement, TMessage>(
  requirement,
  message
);
```

## Generics

- `TInput` <Property {...properties.TInput} />
- `TRequirement` <Property {...properties.TRequirement} />
- `TMessage` <Property {...properties.TMessage} />

## Parameters

- `requirement` <Property {...properties.requirement} />
- `message` <Property {...properties.message} />

### Explanation

With `minProps` you can validate an amount of keys in an object. If the input does not match the `requirement`, you can use `message` to customize the error message.

## Returns

- `Action` <Property {...properties.Action} />

## Examples

The following examples show how `minProps` can be used.

### Minimum properties amount

Schema to validate an object with a minimum amount of 5 properties.

```ts
const MinPropsSchema = v.pipe(
  v.objectWithRest({ id: string() }, v.object({ id: string() })),
  v.minProps(5, 'Object must have at least exceed 5 properties')
);
```
